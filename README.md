## Descripción del Proyecto

Este proyecto es una **página web de BMW** desarrollada con **ReactJS** como framework principal para la materia de Framework de Desarrollo Web. La aplicación presenta una interfaz moderna y responsiva que muestra el catálogo de productos BMW, permite gestionar usuarios, visualizar promociones, ubicaciones de sucursales y más.

### Características Principales

✅ **Catálogo de Productos** - Visualización de vehículos BMW con descripción y precios  
✅ **Carrito de Compras** - Sistema funcional para agregar y eliminar productos  
✅ **Gestión de Usuarios** - Crear, editar y eliminar usuarios con almacenamiento local  
✅ **Galería de Imágenes** - Muestra visual de modelos BMW  
✅ **Mapa de Sucursales** - Localización de tiendas BMW usando geolocalización  
✅ **Información de Clima** - Widget de clima en tiempo real  
✅ **Formularios Responsivos** - Contacto, registro de productos y más  
✅ **Diseño Responsivo** - Compatible con dispositivos móviles y desktop

## Tecnologías Utilizadas

- **ReactJS** - Librería principal de JavaScript
- **Vite** - Build tool y dev server
- **CSS3** - Estilos personalizados y responsive design
- **LocalStorage** - Persistencia de datos en el navegador
- **Fetch API** - Consumo de APIs externas (FakeStore API)
- **Axios** - Cliente HTTP para peticiones

## Instalación y Uso

### Requisitos Previos
- Node.js (v14 o superior)
- npm o yarn

### Instalación

```bash
cd my-react-app
npm install
```

### Ejecución en Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

### Build para Producción

```bash
npm run build
```
